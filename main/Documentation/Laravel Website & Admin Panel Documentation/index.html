A) Overview - top

Otrixweb is an e-commerce website built using the Laravel framework, a popular PHP framework for web development. The website provides a platform for users to buy and sell products online, with features such as product listings, shopping cart, checkout, and payment processing. The website has a modern and user-friendly interface that makes it easy for users to navigate and find the products they are looking for. The product listings include detailed descriptions, images, and reviews from other users to help buyers make informed purchasing decisions. The shopping cart feature allows users to add products to their cart and proceed to checkout, where they can enter their shipping and payment information. The website integrates with various payment gateways to process transactions securely.

Some of the main Technologies used are

  1. PHP 8.1
  2. Laravel 10.x
  3. Bootstrap 5.x

B) System Requirements - top

To installing Otrix Commerce Laravel website, Your web server must be running PHP 8.0 or higher and Mysql 5.7 or higher. We have tested it with Windows, Mac and Linux but for optimum performance we will recommend to use Linux with Apache webserver. Below are a list of items you should ensure your host can comply with.

  • PHP 8.1+
  • MYSQL 5.7+
  • Composer 2.x+
  • MBString Extension
  • MYSQLi Extension
  • CURL Extension (recommended)

* In most hosting accounts, these extensions are enabled by default. But you should check with your hosting provider.

H)Social Login - top

To implement google login follow below proccess

    A.Google Login

  1. Goto the https://console.cloud.google.com/

  2. Click top left dropdown and create new project


  3. Enter project name and click create

  4. Select project

  5. Make sure your project selected & Click OAuth consent screen

  6. Make sure your project selected & Click OAuth consent screen

  7. Select External

  8. Fillup basic details

  9. Fillup website information & click save and continue

  10. Click add remove scopes

  11. Select Top 3 and click save and continue

  12. Add test users it will help you to test your integration before you go live

  13. Click back to dashboard

  14. Click credentials after that click on create credentials and select OAuth Client ID

  15. Application type web application and most important Authorised redirect URL must be same as laravel .env file variable

  16. Copy credentials

  17. Update value in .env file


    That's it your google login implementation done now

B.Facebook Login

  1. Goto the https://developers.facebook.com/ and click on my apps

  2. Click on create app


  3. Click Allow people to login


  4. Enter app name and click create app


  5. Click setting Basic and fillup details


  6. Click on setting > advance and fillup share redirect domain allow list


  7. Click on product > facebook login setting


  8. Fillup valid OAuth Redirect URL


  9. Copy Client App ID and App Secret from setting > basic


  10. Past value in .env file


    11. Your facebook integration completed
